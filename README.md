# Build on windows
Install Node.JS with npm  Python3 and git an Inno Script 
then run this three commands.

```
cd %TEMP% 
explorer %TEMP% 
curl https://raw.githubusercontent.com/alex5250/coda-io-desktop/main/build.py > build.py
py build.py
```
